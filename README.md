# 🎮 PixelPals – Il social per trovare compagni di gioco

Benvenuti su **PixelPals**, una piattaforma pensata per aiutare gamer a trovare altri giocatori compatibili in base a orari, giochi e piattaforme. 🧑‍🤝‍🧑

---

## 🚀 Features principali

- 🔐 Registrazione/Login con JWT + Google/Discord
- 👤 Creazione profilo con giochi preferiti, skill level, piattaforme e disponibilità
- 🔍 Ricerca utenti per gioco/piattaforma
- 🤝 Sistema di matching (manuale e automatico)
- 💬 Chat base (Socket.io o link esterni)
- 🕓 Stato online in tempo reale
- 🏅 Badge, livelli e rating post-partita

---

## 🛠️ Tech Stack

**Frontend:**

- React ⚛️
- Tailwind CSS 🎨 / Bootstrap

**Backend:**

- Spring Boot + Maven ☕
- MongoDB 🍃 o PostgreSQL(pgAdmin)

**Altri strumenti:**

- JWT per autenticazione 🔑
- Google & Discord OAuth2 👥
- Socket.io per real-time 💬
- Railway/Render per backend, Vercel per frontend
